Offer回顾 - 回馈给论坛的小伙伴们



随着和论坛的小伙伴们一起走过秋招，斩获了梦寐以求的offer。最近我收到了一些小伙伴对Summer的一些咨询，希望能够总结回顾自己一路走来的经历，写给还在奋斗的小伙伴们～

背景 & 经历
海本 + MBB PTA + 金融实习 + 互联网实习，作为背景并不突出的候选人，在从投递简历起就充满了很多的顾虑，也有很多depress的时刻。回顾这一段经历，规划、方法、努力三者缺一不可，希望和我一样，对自己背景不自信的小伙伴，一定要抱着破釜沉舟的理念和努力，一路坚持下来。
. ----
申请流程的第一关 - 简历
相信论坛的小伙伴已经对简历的格式、内容、细节管控都有一定的了解，楼主在此不做赘述。在简历部分，我更多会分享一下自己的观点与心得。
在楼主的简历上，最关键的两点因素就是经历本身与经历的展现。从经历本身而言，最理想的情况，就是既有big name, 又有真正有内容的实习/工作经历。但除此之外，经历的展现也至关重要。如何让自己的经历匹配咨询的关键技能；如何选择一些咨询常用词汇；如何排序自己的经历，能够让面试官最快看到最重要的内容；这些才是简历能否通过的决定因素。

闯过了简历关之后 - 笔试
从论坛的讨论话题，我们可以看到，主要的笔试形式有两种：传统的笔试题 / Problem Solving Game (PSG)。论坛上可以找到传统笔试题的一些题库。楼主收到的是PSG，做起来就是策略游戏的模式，主要考察的是整体策略的规划，和遇到问题之后的解决模式。论坛外已有一些视频/文字资源，可以用来参考题目背景。
-baidu 1point3acres
最关键的流程 - 面试
无论是MBB中哪一家，面试都会由behavioral interview + case interview构成，根据不同阶段会有一定差异(比如Bain的written case)。在这两个部分中，很多的小伙伴会认为决定因素是在case interview部分的表现。但是根据楼主的亲身经历，behavioral interview的部分也同样重要，一定要两个部分都充分准备，才能最大几率的拿到最后的offer。

Behavioral Interview我需要准备什么？
在过往的经验中，楼主也为自己的职位面试过很多候选人，所能够设身处地理解面试官的心态。作为一个面试官，我最关注的就是三个问题：
你的个人能力如何？. 1point 3 acres
你是否真正对申请的岗位感兴趣？
你是否是我愿意一起工作的同事？
这三点的重要程度，会随着工作年限的不同而受到不同的关注度。考察的主要方式就是通过cdd过往的项目经历，和交流过程中感受到的兴趣度做判断。
回到cdd的身份中，我也同样做了三件事情，来练习自己回答behavioral interview的整个过程：
选择难度高、参与度高、有影响力的项目：相比于自己讲故事，面试官会更在意整体的交流感，以及会challenge一些项目细节，所以一定要挑选自己参与度高的故事，并不断打磨！
注意自己的交流方式：除了内容本身，交流时候的神态、语调也同样会影响面试的chemistry。所以在打磨好自己的故事之后，也要有意识的去提升整体的面试好感度。. .и
练习、练习、练习：建议除了找小伙伴一起mock以外，也能找有行业经验的小伙伴/学长学姐，这样的反馈是没有面试经验的case partner所不能够给到的。. 1point 3acres

Case Interview我需要准备什么？
在准备的这段时间里，我也遇到了不同经验值的小伙伴，深刻的感受到了case interview练习的几个阶段，以及interview中的主要攻克点。建议小伙伴能够先辨认自己处在哪个阶段，再针对自己的问题做整体时间线上的规划。
关于interview中的主要攻克点，我认为大体可以分为两个部分：能否成功解题，能否顺利deliver出自己的solution，两者缺一不可。
能否成功解题：这一方面涉及到解题过程、和题目类型这两个维度。解题过程是对 Recap > Clarification > Structuring > Chart / Graph Interpretation > Calculation > Brainstorming > Summarization 这整个flow的熟练掌握。题目类型则是对不同框架类型，不同行业类型的全方面练习覆盖。
能否成功deliver：这一方面与做Presentation的大体概念类似。需要能top-down的先给出overview，然后再根据面试官的反馈dig into值得讨论的分支。此外还有与行为面试同样的，对于交流方式的把控能力。
最后的最后，希望能够再次鼓励一下还在路上的小伙伴们。无论是毕业季，还是换工作，相信都是对你们来说非常重要的节点。希望大家都能想清楚自己想要的，并且为自己作出最合适的规划，付出最大程度的努力！祝大家早日能够拿到想要的offer～
